## Azxcore Bot Discord
### What This is bot?
*This bot is make with discord.js v12*

### How to install?
- install node.js
- clone this repo
- install a package
```
npm install
```
- conguration a bot in [this](
- set your token in `.env` file 
```
TOKEN=YouTokenBot
```
- run the bot
```
npm start
```
